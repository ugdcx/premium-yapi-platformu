# Premium Yapı Platformu

Yapı geliştirme, tadilat, değer artırma ve gayrimenkul danışmanlığı süreçlerini tek merkezden yönetmek için hazırlanmış Next.js tabanlı V1 demo platformu.

## Kapsam

- `/` premium landing sayfası
- `/hizmetler` hizmet alanları
- `/surec` dijital proje kontrol süreci
- `/teklif-al` akıllı ön başvuru akışı
- `/login` rol bazlı giriş yönlendirmesi
- `/client` müşteri proje paneli
- `/admin` Kanban başvuru ve operasyon paneli
- `/field` saha ekibi güncelleme paneli

## V1 Özellikleri

- Tutarlı demo veri hikayesi: Ali Atmaca ve Villa Renovasyon Süreci
- Teklif, revize talebi ve onay deneyimi
- Ödeme planı ve belge yönetimi görünümü
- Fotoğraf tabanlı saha ilerleme akışı
- Müşteri talep ve mesaj sistemi
- Teslim ve garanti takip alanı
- Global header, footer ve Vercel uyumlu build yapısı

## Komutlar

```bash
npm install
npm run dev
npm run build
npm run start
```

## Yayın Notu

Bu sürüm backend, veritabanı, gerçek dosya yükleme veya ödeme entegrasyonu içermez. Kontrollü müşteri sunumu ve ürün doğrulama demosu için hazırlanmıştır.
