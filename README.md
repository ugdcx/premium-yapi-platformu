# BLAAG Construction and Architecture

Anahtar teslim yapı, tadilat, değer artırma ve gayrimenkul danışmanlığı
süreçleri için hazırlanmış statik/demo Next.js arayüzü.

Public yüzey BLAAG'in profesyonel yapı ve tadilat hizmetini anlatır. Müşteriye
özel takip ayrıcalığı, fotoğraflı süreç takibi ve şeffaf uygulama yönetimi demo
verilerle temsil edilir.

## Komutlar

```bash
npm run dev
npm run build
npm run start
```
